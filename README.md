# Обзор
Проект для автоматической сборки учебных Anki-карточек из частотного списка японских слов.
Из csv файла по шаблону [слово,часть речи,чтение каной,частота] берет слова и получает переводы из Jardic. Есть возможность
генерировать чтения слов в формате wav, через ИИ модель Kokoro-82M. Генерируются два набора карточек:

JP - RU (слово - перевод)

RU - JP (перевод - слово)

Планируется, что слова будут подкрепляются примерами, если для конкретного слова они есть на Tatoeba. Пока что, 
примеры есть, но только из статей словаря warodai.

# Использование
Проект использует uv. После клонирования для настройки окружения достаточно написать 
```sh
uv sync
```

Скрипты создания словаря или анки-дека запускаются с main.py:
```sh
uv run --package dictmaker python dictmaker/main.py
```

Тесты можно запустить аналогично:
```sh
uv run --package dictmaker pytest dictmaker/tests/
```

Для работы на Windows требует установленной программы-словаря Jardic. Так же рекомендуется скачать дополнительные словари
Warodai и Yarxi, чтобы для большего кол-ва слов нашлись переводы. На non-win системах dictmaker будет парсить онлайн версию Jardic.

# Источники
NINJAL-LWP - https://tsukubawebcorpus.jp/en - список самых используемых японских слов.

Jardic Pro - https://www.jardic.ru - программа японского словаря

Warodai - https://warodai.ru - японско-русский электронный словарь

ЯРКСИ - https://www.yarxi.ru  - японско-русский словарь иероглифов

Kokoro-82M - https://huggingface.co/hexgrad/Kokoro-82M - ИИ модель для TTS преобразований.

Tatoeba - https://tatoeba.org - примеры предложений на всех языках мира
